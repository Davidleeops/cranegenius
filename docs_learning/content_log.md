# Content Log

Track content assets, case studies, and proof points.

---

## Template

| Field | Value |
|---|---|
| Content ID | CON-YYYY-NNN |
| Date | YYYY-MM-DD |
| Asset Type | LinkedIn post / Case study / Email sequence / TikTok / Other |
| Topic | What the content covers |
| Audience | Who it targets |
| Status | Idea / Draft / Published |
| Notes | Supporting data, file references, CTA |
