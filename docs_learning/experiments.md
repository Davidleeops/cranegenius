## Template

| Field | Value |
|---|---|
| Experiment ID | E-YYYY-NNN |
| Date | YYYY-MM-DD |
| Hypothesis | If we do X then Y will happen |
| Change Made | Exact change applied |
| Result | Measured outcome vs baseline |
| Decision | Keep / Revert / Iterate |
| Lesson | Linked Lesson ID or inline note |
