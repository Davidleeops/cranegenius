# Signal Intelligence Log

Append new entries with: python3 scripts/append_signal_log.py

---

## Template

| Field | Value |
|---|---|
| Signal ID | SIG-YYYY-NNN |
| Date | YYYY-MM-DD |
| Signal Type | Permit / SEC Filing / Job Posting / Interconnection Queue / Other |
| Industry | Construction / Data Centers / Industrial / Utilities / Other |
| Source | API endpoint or agency name |
| Observation | What the signal showed |
| Why It Matters | How this predicts crane demand |
| Follow-Up | Next action or status |
